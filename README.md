# FinanceFlow 💸
A sleek, professional finance tracking application built with **React** and **Tailwind CSS**. 

## 🚀 Live Demo
[View the Live App here!](https://simon-dawit.github.io/for_portfolio/)

## ✨ Features
* **Real-time Currency Conversion:** Fetches live rates via the ExchangeRate-API.
* **Base Currency Selection:** Track your spending in USD, EGP, EUR, and more.
* **Multi-language Support:** Interfaces in English, Tigrigna, Amharic, and more.
* **Persistent Storage:** Uses `localStorage` to save your transactions.
* **Dark Mode:** Gritty, atmospheric UI for late-night budgeting.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS
* **API:** Open Exchange Rates API
* **Icons:** Heroicons / SVG
